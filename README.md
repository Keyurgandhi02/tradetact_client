# 📊 TradeTact — Trading Analytics & Journal Platform

TradeTact is a full-stack trading platform designed to help traders **track, analyze, and improve their performance** using data-driven insights instead of emotions.

Built with a **React Native mobile app** and a **web-based admin panel**, TradeTact provides complete control over trading data, user management, and analytics.

---

## 🚀 Overview

Most traders lose money not because of strategy, but because of:
- Lack of discipline  
- No tracking system  
- Emotional decision-making  

TradeTact solves this by:
👉 Turning trading into a **data-driven process**  
👉 Helping users track behavior and performance  
👉 Providing insights to improve consistency
